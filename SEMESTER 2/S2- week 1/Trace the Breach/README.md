# Trace the Breach: Incident Response Simulation

This folder contains materials related to the "Trace the Breach" exercise, an incident response simulation undertaken during Week 1 of Semester 2. It provides practical experience in identifying, analyzing, and responding to security incidents.

## Contents
- `Incident_Response_Summary_Template.pdf`: Template used for summarizing incident responses.
- `Incident_Response_Worksheet.pdf`: Worksheet to guide the incident response process.
- `Instructor Notes.pdf`: Notes provided by the instructor for the exercise.
- `README_STUDENT.rtf`: Student-facing instructions for the exercise.
- `incident_scenario.pdf`: Description of the simulated incident scenario.
- `phishing_email_example.pdf`: Example of a phishing email.
- `security_logs.rtf`: Simulated security logs for analysis.
- `slack_screenshots.pdf`: Screenshots from a simulated communication channel.

## Purpose
The primary purpose of this exercise was to develop critical thinking and practical skills in cybersecurity incident response, a key component of workforce readiness in the tech field.
