DAE – Web Dev (Back-End) Intro Activity

SETUP:
1. Copy this folder to /Applications/MAMP/htdocs/
2. Start MAMP and launch servers
3. Visit http://localhost/phpmyadmin
4. Create a DB named: webdev_intro
5. Import db.sql

RUN:
- Open http://localhost/webdev-backend-intro/index.html
- Fill and submit the form
- View messages and database
- Try edit/delete options
