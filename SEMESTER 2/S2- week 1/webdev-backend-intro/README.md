# Web Development Backend Introduction

This folder contains introductory materials and exercises for backend web development, part of Week 1 of Semester 2. It covers fundamental concepts and practical implementation of server-side logic.

## Contents
- `README_INSTRUCTIONS_EDUCATORS.txt`: Instructions for educators (for context).
- `README_STUDENT.txt`: Student instructions for the web development exercises.
- `db.sql`: Database schema or initial data.
- `edit.php`, `form_handler.php`, `index.html`, `view.php`: Files comprising a simple web application demonstrating backend functionality.

## Purpose
This module aimed to introduce the basics of backend web development, including database interaction and server-side scripting, providing a foundational understanding for building dynamic web applications.
