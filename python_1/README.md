# EcoStyle -- Sustainable Fashion Selector

## Project Description

EcoStyle is a simple Python command-line application that helps users
choose sustainable fashion options based on different conditions. The
program asks the user for the **occasion, weather, and preferred
fabric**, then processes the information and provides suggestions.

This project demonstrates **basic Python concepts such as user input,
conditional statements, and program flow**.

------------------------------------------------------------------------

## Features

-   User-friendly command line interaction
-   Fashion suggestions based on **occasion**
-   Weather-based clothing selection
-   Fabric preference input
-   Basic error handling for incorrect inputs

## Example Program Flow

The program asks the user to enter:

1.  Occasion
    -   Wedding
    -   Party
    -   Casual
2.  Weather
    -   Hot
    -   Mild
    -   Cold
3.  Fabric Preference
    -   Cotton
    -   Silk
    -   Jute

After collecting the inputs, the program processes the information and
displays suggestions.
