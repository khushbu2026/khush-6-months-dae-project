# Logic Fundamentals

This folder contains materials and exercises related to logic and algorithmic thinking. This is where foundational concepts for problem-solving and programming logic are explored.

## Contents
- `dice algorithm and flow chart.pdf`: Documentation on a dice algorithm and its flowchart.
- `dice_game.py`: A Python implementation of a dice game, likely used to demonstrate logical constructs.

## Purpose
The purpose of this folder is to build and strengthen logical reasoning and problem-solving skills, which are essential for any programming or technical role.
